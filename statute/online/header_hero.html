前言

《簡明法規彙編》為臺灣大學開源社於 105 學年度第 2 學期首度發行之法規工具書。透過解析學生代表大會秘書處用以刊印正式法規彙編的原始條文與學生法院解釋，轉換成文件物件模型後，再依據 CSS3 分頁媒體模組標準產生可以線上閱覽、並即可列印 (print-ready) 的 HTML 文件；

簡單來說，就是透過程式精簡條文，用「網頁」這種大家習以為常的開放標準排版，並產生具有印刷水平、又同時可以線上閱讀的「民間版」法典。除發行紙本版本外，亦可於此網站瀏覽線上版本；用以解析、排版之程式原始碼，亦於 GitHub 上以 GNU GPLv3 公眾授權釋出。
